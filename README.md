# MERN-Ecommerce
MERN stack shop application

## Installation

install packages
```bash
cd backend
npm install
cd frontend
npm install
```
create env file
```env
PORT=...
MONGO_URI=...
SECRET=...
```
change constant apiUrl in frontend
```js
const apiUrl = 'Your api url';
export default apiUrl;
```

## Usage

```bash
cd backend
npm run dev

cd frontend
npm run dev
```
